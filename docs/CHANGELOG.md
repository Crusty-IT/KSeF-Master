# Changelog

Wszystkie istotne zmiany w projekcie KSeF Master będą dokumentowane w tym pliku.
Format jest oparty na [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-12-30
### Added
- Pełna dokumentacja projektu w folderze `docs/`.
- Widok logowania z walidacją NIP i Tokena KSeF.
- Pulpit nawigacyjny (Dashboard) z podsumowaniem faktur.
- Moduł wystawiania nowych faktur.
- Lista faktur odebranych i wystawionych z integracją API.
- Generowanie raportów finansowych.
- Eksport faktur do formatu PDF.
- Zarządzanie bazą kontrahentów.

### Changed
- Odświeżony interfejs użytkownika z ciemnym motywem (dark mode).
- Usprawniona nawigacja boczna (SideNav).

### Fixed
- Poprawka obsługi błędów podczas logowania do bramki KSeF.
- Naprawa formatowania kwot walutowych w podglądzie faktury.
